Aide mémoire des commandes de base Git

Cloner un repo

Cloner un repo Git permet de créer une copie exacte d'un repo remote (distant) sur votre machine locale.

git clone REPO_URL

Remplacer REPO_URL par l'adresse de votre repo.

Commit

L'opération Commit permet de renseigner les modifications apportées au code.

git commit -m "Message à renseigner en lien avec la modification"

Vous remarquerez, avec le temps, que certains utilisent des conventions d'écriture.

Par exemple, si le commit est:

  • un ajout de code, on préfixe par ADD
  • une correction de bogue, on préfixe par FIX
  • un travail en cours, on préfixe par WIP
